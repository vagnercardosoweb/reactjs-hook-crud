# Inciar backend

Utilizado para backend o [JSON Server](https://github.com/typicode/json-server)

```bash
cd backend
yarn install && yarn start
# npm install && npm run start
```

# Inciar frontend

Utilizado para frontend o [ReactJS](https://reactjs.org/)

```bash
cd frontend
yarn install && yarn start
# npm install && npm run start
```
